# Project Notes

## UI Style

**风格**: 苹果玻璃
**来源**: built-in
**描述**: 以完整玻璃应用窗口为骨架，使用冷青与清蓝的双色流体背景建立空间张力，通过透明度、模糊、亮边和留白构建高级且克制的聊天工具界面。
**参考文件**: references/glass.md
**风格契约**:
- 场景: 实时聊天工具与后台管理界面
- 气质: 通透、秩序、克制、精密、轻柔
- 布局: 全屏居中的应用窗口，侧栏加主内容区，局部吸顶头部
- 排版: 人文无衬线，标题紧凑，正文疏朗，用字重和透明度建立层级
- 配色/材质: 冷雾白基底，冷青与清蓝双色流体背景，玻璃半透明层、亮边、内阴影与细颗粒
- 动效: 缓慢漂移背景，轻微提亮与上浮，避免弹跳和高频动效
- 禁忌: 禁止蓝紫渐变、禁止 emoji 图标、禁止大面积纯白卡片堆叠、禁止无信息装饰组件
